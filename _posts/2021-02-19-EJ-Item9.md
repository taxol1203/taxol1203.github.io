## 자바의 두 가지 객체 소멸자
### finalizer
```java
@Override
public void finalize() {
    // ...
}
```
finalize 메서드를 Override하면 해당 객체가 JVM에게 GC를 해야 할 대상이 될 때 호출된다.  
객체가 없어지기 전 다른 연관 자원을 정리하려는 의도로 작성된다.  

하지만 예측할 수 없고, 상황에 따라 위험할 수 있어 일반적으로 불필요하다.
